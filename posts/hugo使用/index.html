Hugo使用

Posted on Mar 3, 2020

简介

​ Hugo是一种静态网站生成器。适用于搭建个人博客、小型公司主页等网站，是一种小型的CMS系统。静态站点的好处就是快速、安全、易于部署，方便管理。

​ 在csdn或者有道笔记等平台上进行个人知识管理，很多地方受限制，还收费啥啥的，也不知道几年后会不会关门了。所以折腾下个人博客也挺好的，记录些自己平时的技术经历。

hugo优势

快！！！世界上最快的静态网站生成工具！5秒生成6000个页面！
文档为Markdown格式,语法超简单；
丰富的主题；
多环境支持：macos、linux、windows

安装

golang安装

yum -y install golang

hugo安装

添加源

vim /etc/yum.repos.d/hugo.repo

[daftaupe-hugo]
name=Copr repo for hugo owned by daftaupe
baseurl=https://copr-be.cloud.fedoraproject.org/results/daftaupe/hugo/epel-7-$basearch/
type=rpm-md
skip_if_unavailable=True
gpgcheck=1
gpgkey=https://copr-be.cloud.fedoraproject.org/results/daftaupe/hugo/pubkey.gpg
repo_gpgcheck=0
enabled=1

安装hugo

yum -y install hugo

版本查看

hugo version

本地博客搭建

创建站点

[root@localhost hugo]# hugo new site mysite

[root@localhost hugo]# cd mysite/
[root@localhost mysite]# ls -l
总用量 4
drwxr-xr-x 2 root root 24 3月   3 01:13 archetypes
-rw-r--r-- 1 root root 82 3月   3 01:13 config.toml
drwxr-xr-x 2 root root  6 3月   3 01:13 content
drwxr-xr-x 2 root root  6 3月   3 01:13 data
drwxr-xr-x 2 root root  6 3月   3 01:13 layouts
drwxr-xr-x 2 root root  6 3月   3 01:13 static
drwxr-xr-x 2 root root  6 3月   3 01:13 themes

目录作用：
    archetypes：存放default.md，头文件格式 
    config.toml：config.toml是网站的配置文件
    content：包括网站内容，全部使用markdown格式
    data：存放自定义模版，导入的toml文件（或json，yaml） 
    layouts：包括了网站的模版，决定内容如何呈现
    static：包括了css, js, fonts, media等，决定网站的外观
    themes：存放网站主题文件

添加主题

[root@localhost mysite]# git init
[root@localhost mysite]# mkdir themes
[root@localhost mysite]# cd themes
[root@localhost mysite]# git clone https://github.com/keichi/vienna

编写文章

[root@localhost mysite]# hugo new "posts/hello-world.md"  
[root@localhost mysite]# vi content/posts/hello-world.md

运行服务

[root@localhost mysite]# hugo server -D --theme=vienna --buildDrafts

访问博客

http://localhost:1313/

github线上部署

创建仓库

登陆github网站
创建仓库，仓库名称为 用户名.github.io

配置修改

[root@localhost mysite]# vi config.toml
baseURL = "https://xiaofeng-zerone.github.io/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "vienna"

public生成

[root@localhost mysite]# hugo --theme=vienna --buildDrafts --baseUrl="https://xiaofeng-zerone.github.io/"

public推送

git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/xiaofeng-zerone/xiaofeng-zerone.github.io.git
git push -u origin master

在线访问

https://xiaofeng-zerone.github.io/
